6. Anillos centralizadores (conmutadores)¶

6.1. Encontrar una base para el centralizador¶

6.1.1. CentralizerBlocksOfRepresentation¶

La función CentralizerBlocksOfRepresentation( rho ) devuelve una lista de generadores de espacio vectorial para el anillo centralizador de \(\rho(G)\), escrito en la base dada por BlockDiagonalBasisOfRepresentation. Las matrices se dan como una lista de bloques.

Sea \(G\) representaciones irreductibles \(\rho_{i}\) con multiplicidades \(m_{i}\). El centralizador tiene una dimensión \(\displaystyle\sum_{i} m_{i}^{2}\) como un \(\mathbb{C}-\text{espacio vectorial}\). Esta función proporciona el número mínimo de generadores necesarios.

6.1.2. CentralizerOfRepresentation¶

La función CentralizerOfRepresentation( arg ) devuelve una lista de generadores de espacio vectorial para el anillo centralizador de \(\rho(G)\).

Esto da el mismo resultado que CentralizerBlocksOfRepresentation, pero con las matrices dadas en su totalidad: no como listas de bloques, sino como matrices completas.

6.2. Usando el centralizador para diversos cálculos¶

6.2.1. ClassSumCentralizer¶

La función ClassSumCentralizer( rho, class, cent_basis) devuelve \(\displaystyle\sum_{s \in t^{G}} \rho(s)\), donde \(t\) es un representante de la clase de conjugación clase de \(G\).

Requerimos que rho sea unitario. Utiliza la base ortonormal dada (con respecto al producto interno \(\langle A, B \rangle = \text{Trace}(AB^\ast)\) para el anillo centralizador de rho para calcular la suma de la clase de conjugación class rápidamente, es decir, sin sumar la clase.

Nota

  • Se comprueban la ortonormalidad de cent_basis y la unitaridad de rho. Consulte ClassSumCentralizerNC para obtener una versión de esta función sin comprobaciones. Las verificaciones no son muy costosas, por lo que se recomienda utilizar la función con chequeos.

6.2.2. ClassSumCentralizerNC¶

La función ClassSumCentralizerNC(rho, class, cent_basis)

Lo mismo que ClassSumCentralizer, pero no verifica la base de la ortonormalidad o la representación de la unitaridad. Ya sabemos que NC significa «No check».
